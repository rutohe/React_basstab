# React Basstab
Reactを使って開発した、ブラウザ上で直感的にベースのタブ譜を作成・編集できるWebアプリケーションです。
「苦手なフレーズを繰り返し練習したい」「自分だけのオリジナル運指集を作りたい」という目的から生まれ、ドラッグ＆ドロップを中心とした高速な入力UXを実現しています。
## コンセプト・注意
リズム等の記号はよくわからない＆現状、技術的に実装が難しいので、
## 実装機能
- 編集用のモーダル
- 譜面のオブジェクト表示
- ドラッグでのフレット数入力
- ハンマリング、プルオフの表示
## 実装予定機能
- Ruby on Railsを用いて譜面を保存、引き出しができるようにする
## 開発目的
- Reactの学習
- 苦手なフレーズを繰り返し練習できるようにするため
- 自分だけの好きなフレーズや苦手フレーズを集めたタブ譜集を作るため
- 将来的にRubyを使ってほかのユーザーの譜面を入手できるようにして、弾きたい曲の譜面を見つけやすくしたいため
## 使用技術
- HTML
- CSS
- JavaScript
- React
## データ構造案
```js
column = {
    fret:[1,null,null,null],    //-1(ミュート)と0(開放弦)~21まで
    hammer_on:true,             //ハンマリング
    pull_off:false,             //プルオフ
    bar:false,                  //小節線か否か
}
row = [column,column,...];      //columnの集合で譜面を表現
```
## コンポーネント分割案
#### App
- state管理
- データ保存
- 譜面全体表示
#### Row
- column並べて表示
#### Column
- 1列表示
#### Modal
- columnへの入力受付
- 必要ならUIからデータに変換
